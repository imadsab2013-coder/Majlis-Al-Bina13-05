## 1️⃣2️⃣ سلسلة الاستدعاءات (Call Chain)

```
streamlit run app_ui.py
         │
         ↓
main script execution
         │
         ├─→ st.set_page_config()
         ├─→ CSS injection
         ├─→ initialize_session()
         ├─→ AGENTS dictionary loading
         ├─→ Function definitions (check_data_files, etc.)
         └─→ UI Rendering:
             │
             ├─→ st.markdown (Header)
             │   └─→ Agent Bar Rendering
             │       └─→ For each agent: st.button()
             │
             ├─→ col_main, col_sidebar = st.columns()
             │
             ├─→ Board Container
             │   └─→ Display board_output items
             │
             ├─→ Input Section
             │   └─→ st.text_area() + st.button()
             │       └─→ on click: append to board_output
             │           └─→ st.rerun()
             │
             └─→ Sidebar
                 ├─→ API Settings Expander
                 │   └─→ st.text_input() + st.slider()
                 │       └─→ on "Connect": update_connection_status()
                 │
                 ├─→ Rules Manager Expander
                 │   └─→ st.text_input() + st.text_area()
                 │       └─→ on "Add": append to rules list
                 │
                 └─→ Session Management
                     └─→ on "Save": write JSON file
```
