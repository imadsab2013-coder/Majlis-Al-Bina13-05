# 📖 مجلس البينة | Majlis Al-Bina

## نظام تحليل النص القرآني الذكي
**Intelligent Quranic Text Analysis System**

---

## 🚀 كيفية التشغيل

### 1. التثبيت الأولي

```bash
# تثبيت المكتبات المطلوبة
pip install -r requirements.txt
```

### 2. إعداد ملفات البيانات

```bash
# إنشاء مجلد data
mkdir data

# ضع الملفات التالية في المجلد:
# - data_quran.xlsx
# - data_words.xlsx
```

### 3. تشغيل التطبيق

```bash
streamlit run app_ui.py
```

سيفتح المتصفح على `http://localhost:8501`

---

## 📋 مكونات الواجهة

### 🎯 شريط الأيقونات العلوي (Top Agent Bar)
- **10 أيقونات دائرية**: تمثل الأعضاء من 1A إلى 10A
- **مؤشر الحالة 🟢🔴**: يدل على حالة اتصال كل عضو
  - 🟢 أخضر: متصل وجاهز
  - 🔴 أحمر: غير متصل
  - 🟡 أصفر (للآمر 8A): حالة خاصة

### 📊 السبورة (The Board)
- عرض النتائج بشكل متسلسل
- كل نتيجة تظهر في **بلوك منفصل** يحمل اسم الوكيل ورقمه
- يمكن مسح السبورة أو حفظها

### 📝 صندوق الإدخال (Input Box)
- كتابة الاستفسار أو الملحوظة
- إرسال الطلب للنظام

### ⚙️ الشريط الجانبي (Sidebar)

#### 🔑 إدارة API
- إدخال مفتاح Gemini API
- اختيار الموديل (Gemini / GPT)
- ضبط درجة الحرارة (Temperature)
- زر اختبار الاتصال

#### 🟢 حالة الاتصال
- حالة اتصال API
- حالة تحميل ملفات البيانات

#### 📋 إدارة القواعس §
- إضافة قواعد جديدة
- عرض القواعس المفعلة
- حذف القواعس

#### 💾 إدارة الجلسة
- حفظ الجلسة الحالية
- استيراد جلسات سابقة
- تصدير البيانات
